文档说明
********

这些文档是用 Sphinx 从 reStructuredText 源生成的，*Sphinx* 是一个专为
处理 Python 文档而编写的文档生成器。

本文档及其工具链之开发，皆在于志愿者之努力，亦恰如 Python 本身。如果您
想为此作出贡献，请阅读 处理错误 了解如何参与。我们随时欢迎新的志愿者！

特别鸣谢：

* Fred L. Drake, Jr.，原始 Python 文档工具集之创造者，众多文档之作者；

* Docutils 项目，该项目孕育了 reStructuredText 及 Docutils 套件；

* Fredrik Lundh 的项目 Alternative Python Reference，Sphinx 从中得到了
  很多不错的点子


Python 文档的贡献者
===================

有很多对 Python 语言，Python 标准库和 Python 文档有贡献的人，随 Python
源代码发布的 Misc/ACKS 文件列出了部分贡献者。
